Summary Findings: Latitudes vs Cloudiness


We can see plainly here that it's possible to find 100% cloudy days at any latitude, but it there is a much lower liklihood of clear skies between -20 to 0 degrees latitude. These are the only two things we can really gather from this data, as all cloud patterns can be found at any latitude.
